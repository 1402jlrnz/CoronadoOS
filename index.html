What is an Operating System?

Definition

An Operating System (OS) is system software that manages computer hardware, software resources, and provides common services for computer programs. It acts as an intermediary between users and computer hardware.

Resource Management

Manages CPU time, memory, storage, and input/output devices

Process Management

Handles creation, scheduling, and termination of processes

Memory Management

Allocates and deallocates memory space to programs

File System

Organizes and manages files and directories

User Interface

Provides interface for users to interact with the computer

Security & Protection

Protects system resources and ensures data security

Examples of Operating Systems

  • Windows: Microsoft Windows 10, Windows 11
  • macOS: Apple's operating system for Mac computers
  • Linux: Ubuntu, Fedora, Debian, Arch Linux
  • Mobile: Android, iOS
  • Server: Windows Server, Linux distributions

Windows 95 Installation

1. Create a new VirtualBox VM

  • Type: Microsoft Windows
  • Version: Windows 95
  • RAM: 32 MB
  • Disk: 500 MB

2. Attach Windows 95 ISO

Mount the Windows 95 installation ISO file to the virtual machine's CD/DVD drive.

3. Start VM and boot from ISO

Power on the virtual machine and ensure it boots from the attached ISO file.

4. Follow on-screen setup steps

Complete the Windows 95 installation wizard following the prompts.

5. Restart to finish installation

Reboot the system to complete the Windows 95 setup process.
