What is an Operating System?

Definition

An Operating System (OS) is system software that manages computer hardware, software resources, and provides common services for computer programs. It acts as an intermediary between users and computer hardware.

Resource Management

Manages CPU time, memory, storage, and input/output devices

Process Management

Handles creation, scheduling, and termination of processes

Memory Management

Allocates and deallocates memory space to programs

File System

Organizes and manages files and directories

User Interface

Provides interface for users to interact with the computer

Security & Protection

Protects system resources and ensures data security

Examples of Operating Systems

  • Windows: Microsoft Windows 10, Windows 11
  • macOS: Apple's operating system for Mac computers
  • Linux: Ubuntu, Fedora, Debian, Arch Linux
  • Mobile: Android, iOS
  • Server: Windows Server, Linux distributions

Adding Users and Logging In 5 Person Per Section

1. Create Organizational Units

Open Active Directory Users and Computers → Create separate OUs for each department or section (e.g., Sales, Marketing, IT, HR, Finance).

2. Create User Accounts

Right-click each OU → New → User → Create 5 user accounts per section with proper naming convention and initial passwords.

3. Configure User Properties

Set user profile information, group memberships, and password policies for each user account.

4. Set Up Login Scripts

Configure login scripts or group policies to map network drives and set up desktop environments for each user.

5. Test User Logins

Verify each user can successfully log in to their assigned workstation with their credentials and access appropriate resources.

DHCP Role and Joining Workstation in Domain

1. Install DHCP Role

Open Server Manager → Add Roles and Features → Select DHCP Server → Complete installation wizard.

2. Configure DHCP Scope

Create new scope with IP range, subnet mask, default gateway, and DNS server settings.

3. Set Up Active Directory

Install Active Directory Domain Services role and promote server to domain controller.

4. Create Domain Users

Use Active Directory Users and Computers to create user accounts and organizational units.

5. Join Workstation to Domain

On workstation: System Properties → Change → Enter domain name → Provide credentials → Restart.

RAID Configuration

1. Prepare Storage Disks

Ensure multiple disks are available and properly connected to the server.

2. Open Disk Management

Right-click Computer → Manage → Disk Management or use Server Manager storage tools.

3. Create RAID Array

Select disks → Right-click → Create New Spanned/Stripped/Mirrored Volume based on RAID type needed.

4. Configure RAID Level

Choose appropriate RAID level: RAID 0 (striping), RAID 1 (mirroring), or RAID 5 (parity).

5. Format and Initialize

Assign drive letter, format with NTFS, and set volume label for the new RAID array.

Group Policy

2. Create New GPO

3. Configure Policy Settings

5. Test Policy Application
